Install bookmarklet

Drag the button to your bookmarks bar (desktop), or copy the code and paste it into a new bookmark (mobile).

Drag this to your bookmarks bar 

      
Tip: You can rename the bookmark to anything you like after adding it.

    


    

      
Desktop install

      
    
        
  • Show your bookmarks bar if it’s hidden:
          
    
            Chrome/Edge: Ctrl+Shift+B (Windows/Linux) or Cmd+Shift+B (macOS) • Firefox: Ctrl+B or Cmd+B • Safari: View → Show Favorites Bar
          
    
        
    • 
        
  • Drag the blue “Drag this…” button into the bar.
    • 
        
  • Done. Click the new bookmarklet to run it on any page of the same site.
    • 
      

    


    

      
Mobile install

      
    
        
  • Tap “Copy code”.
    • 
        
  • Create a new bookmark on any page.
    • 
        
  • Edit the bookmark and paste the code into the URL/location field (replace the page URL).
    • 
        
  • Save. Use it from your bookmarks when you need it.
    • 
      

      
iOS Safari: Use the Share button → Add Bookmark (not “Add to Home Screen”). Then edit the bookmark and paste the code into the address field.

    


    

      
What this bookmarklet does

      
This example prompts for a key and value and saves it to localStorage on the current site. Replace the code with your own if needed.

      

        Storage: localStorage
        Scope: current site
        Persistent
